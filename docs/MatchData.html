Overview

Represents a match data object that provides methods to convert match data into other formats such as a hash or an OpenStruct.

This class extends the functionality of the standard MatchData class in Ruby.

Examples:

Converting MatchData to OpenStruct

match_data = /(\w+)/.match("hello")
struct = match_data.to_struct
puts struct[0] # => "hello"

#to_hash ⇒ Hash

Converts the MatchData object to a hash.

The hash keys are the names of the captures, and the values are the corresponding capture strings, stripped of whitespace. If a capture can be converted to an integer, it will be returned as an integer.

Examples:

Converting MatchData to Hash

match_data = /(\d+)/.match("123")
hash = match_data.to_hash
puts hash # => {"0"=>"123", "1"=>123}

Returns:

  • (Hash) —

    a hash representation of the match data, with capture names as keys.

def to_hash
  Hash[self.names.zip(self.captures.map(&:strip).map do |capture|
    if capture.is_i? then capture.to_i else capture end
  end)]
end
