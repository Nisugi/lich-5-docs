Overview

In-memory cache with database read fallbacks.

This class provides a simple in-memory cache that allows for storing, retrieving, and managing key-value pairs. It also supports fetching values from a block if the key is not present in the cache.

Examples:

Basic usage

cache = Lich::Gemstone::Infomon::Cache.new
cache.put(:foo, 'bar')
cache.get(:foo) # => 'bar'

def get(key)
  return @records[key] if self.include?(key)
  miss = nil
  miss = yield(key) if block_given?
  # don't cache nils
  return miss if miss.nil?
  @records[key] = miss
end
