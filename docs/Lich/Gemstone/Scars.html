Overview

Scars class for tracking character scars

This class provides methods to access and manipulate the scar data of a character's body parts. It uses XML data from Simutronics to structure the scar information.

Examples:

Accessing scar levels

scar_level = Lich::Gemstone::Scars.leftEye
all_scar_levels = Lich::Gemstone::Scars.all_scars

def all_scars
  begin
    fix_injury_mode('scar') # for this one call, we want to get actual scar level data
    result = XMLData.injuries.transform_values { |v| v['scar'] }
  ensure
    fix_injury_mode('both') # reset to both
  end
  return result
end

def arms
