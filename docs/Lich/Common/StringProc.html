Overview

Represents a String processing object that can evaluate a string as a Ruby expression. This class allows for the creation of callable objects that encapsulate a string and can execute it.

Examples:

Creating a StringProc and calling it

sp = Lich::Common::StringProc.new('1 + 1')
sp.call # => 2
