Overview

A thread-safe wrapper around a socket object that synchronizes access to the underlying socket methods using a mutex.

This class is useful in multi-threaded environments where multiple threads may attempt to read from or write to the same socket.

Examples:

Creating a synchronized socket

socket = TCPSocket.new('localhost', 8080)
synchronized_socket = Lich::Common::SynchronizedSocket.new(socket)

def puts_if(*args)
  @mutex.synchronize {
    if yield
      @delegate.puts(*args)
      return true
    else
      return false
    end
  }
end
