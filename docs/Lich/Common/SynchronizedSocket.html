Overview

Represents a thread-safe wrapper around a socket. This class ensures that socket operations are synchronized using a mutex.

Examples:

Creating a synchronized socket

socket = SynchronizedSocket.new(original_socket)

def puts_if(*args)
  @mutex.synchronize {
    if yield
      @delegate.puts(*args)
      return true
    else
      return false
    end
  }
end
