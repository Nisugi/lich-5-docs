.to_hash ⇒ Object

Returns a hash-like object representing the current game settings.

Note: This method does not behave like a standard Ruby hash request. It returns a root proxy for the game settings scope, allowing persistent modifications on the returned object for legacy support.

Examples:

Getting game settings as a hash-like object

settings_hash = Lich::Common::GameSettings.to_hash

Returns:

  • (Object) —

    a proxy object for the game settings
