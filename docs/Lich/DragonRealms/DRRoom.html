Overview

Represents a room in the DragonRealms game.

This class manages various aspects of a room, including NPCs, PCs, group members, and room descriptions. It provides class-level accessors for managing the state of the room.

Examples:

Creating and accessing room data

room = DRRoom.new
DRRoom.npcs = [npc1, npc2]
puts DRRoom.npcs
