Overview

The Flags class manages a set of flags and their associated matchers.

This class provides methods to add, reset, delete, and access flags and their matchers.

Examples:

Usage

Flags.add(:example_flag, 'example')
Flags[:example_flag] # => false
Flags[:example_flag] = true
Flags.reset(:example_flag)
Flags.delete(:example_flag)

def self.add(key, *matchers)
  @@flags[key] = false
  @@matchers[key] = matchers.map { |item| item.is_a?(Regexp) ? item : /#{item}/i }
end
